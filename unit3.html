Unit 3: Graph Algorithms

Overview of Topics

This unit covers key graph algorithms, such as Dijkstra’s Algorithm, DFS, and BFS. Understanding these algorithms will enable you to solve real-world network problems.

BFS explores nodes layer by layer from a starting point, ideal for finding the shortest path in unweighted graphs.

DFS dives deep into each branch before backtracking, making it useful for detecting cycles and finding connected components.

Dijkstra's Algorithm finds the shortest path between nodes in a weighted graph with non-negative weights, commonly used in GPS and network routing.

Bellman-Ford is a shortest-path algorithm that can handle graphs with negative weights, unlike Dijkstra’s Algorithm.

Kruskal’s Algorithm finds a minimum spanning tree (MST) for a graph, ensuring that all vertices are connected with minimal edge weight.
